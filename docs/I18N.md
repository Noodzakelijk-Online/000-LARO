# Internationalization — Dutch/English (Phase 057)

Date: 2026-07-06 · Branch `Phase-Imp`

`shared/i18n.ts` provides a dependency-free foundation:
- a message catalog with `nl` + `en` for each key,
- `t(key, locale, vars)` with `{var}` interpolation and fallback
  (requested locale → English → the key),
- `normalizeLocale()` / `isLocale()` helpers; default locale `nl`.

The legal disclaimer (Phase 013) already ships NL + EN.

## Status
**Partial**: the foundation + catalog + tests are in place. Replacing every
hardcoded renderer string with `t()` is an incremental follow-up (paired with the
component-test harness). `tests/backend/i18n.test.ts` verifies translation,
fallback, interpolation, and that every catalog entry has both languages.
